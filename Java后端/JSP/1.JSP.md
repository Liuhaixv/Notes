# JSP

#### HelloWorld示例

```jsp
<%@ page contentType="text/html;charset=UTF-8" language="java" pageEncoding="UTF-8" %>
<!DOCTYPE html>
<html lang="zh">
<head>
    <meta charset="UTF-8">
    <title>HelloWorld</title>
</head>
<body>
<%
    out.println("HelloWorld");
%>
</body>
</html>
```

#### 优点

* 可以编写java代码,使用灵活

#### 缺点

* 耦合性高,后期难于维护
